GWBrowser Documentation¶

GWBrowser is a simple digital asset manager aimed at small and medium-sized animation and film productions.

It helps you navigate digital productions by creating and annotating the files and folders used to keep projects. To learn more on what GWBrowser does and how you can have a look at the Bookmarks & Assets or Python Modules Documentation.

Get GWBrowser¶

You can get the latest version from Github Releases (see the notes about building from source: Installing From Source).
