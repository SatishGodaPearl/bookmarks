.. Add widgets
.. -----------
..
.. GWBrowser can create projects and assets based template files.
.. The templates themselves are simple zip files containing a folder structure.
.. See the **gwbrowser/templates** folder.
..
.. The 3 widgets used to add items are ``AddBookmarksWidget``, ``AddAssetWidget`` and ``AddJobWidget``.
