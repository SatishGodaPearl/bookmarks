**GWBrowser is a simple asset browser used to create, annotate and navigate
the files and folders of animation and film productions.**

.. image:: _static/banner.png
   :alt: GWBrowser

To see how GWBrowser works see :ref:`overview`. There's a bit more information
on the technical side of things at :ref:`modules-documentation`.


.. image:: _static/assets.gif
  :target: _static/assets.gif
  :alt: Assets...



Download
========

You can get the latest version from `Github Releases <https://github.com/wgergely/GWBrowser/releases/>`_
See the notes about building from source: :ref:`installing-from-source`.
